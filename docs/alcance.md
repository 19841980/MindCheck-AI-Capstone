### Alcance del Proyecto: MindCheck AI

El proyecto consiste en una Progressive Web App (PWA) diseñada para el monitoreo preventivo del bienestar emocional para los estudiantes de Duoc UC.



###### **1. Funcionalidades Principales (MVP - Producto Mínimo Viable)**

###### 

* Registro de Bitácora Emocional: Un espacio donde el estudiante escribe cómo se siente. No es solo un "emoji", sino texto libre que la IA procesará.



* Análisis de Sentimiento con IA (NLP): Integración con modelos de procesamiento de lenguaje natural (como los de OpenAI o librerías de Python como Transformers) para detectar niveles de estrés, ansiedad o depresión en el texto.



* Dashboard Personal de Bienestar: Visualización de gráficos de tendencia (ej: "Tu nivel de estrés ha subido un 15% esta semana").



* Sistema de Alertas Inteligentes: Si la IA detecta un patrón de riesgo alto, el sistema enviará una notificación al usuario sugiriendo contacto con el área de apoyo estudiantil.



* Recursos de Autoayuda Personalizados: Sugerencias automáticas de ejercicios de respiración o técnicas de estudio basadas en el estado anímico detectado.







###### **2. Alcance Técnico (Lo que se va a construir)**

|***Área***|***Tecnología Propuesta***|
|-|-|
|**Frontend**|React.js o Next.js (para que sea rápido y móvil).|
|**Backend**|Python con FastAPI o Flask (ideal para manejar la IA).|
|**IA/ML**|Modelos pre-entrenados de Hugging Face o GPT-4o mini API.|
|**Base de Datos**|PostgreSQL o MongoDB para guardar los registros de forma segura.|
|||



###### **3. Límites del Proyecto (Lo que NO incluye)**





* El sistema no reemplaza a un psicólogo profesional; es una herramienta de monitoreo preventivo.



* No se integrará con fichas clínicas médicas externas (por temas de privacidad legal compleja).
